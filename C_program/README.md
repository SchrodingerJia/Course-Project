# C语言学习项目

## 项目概述
本项目为C语言课程学习材料，包含作业与实验两部分内容。

## 目录结构

### 作业 (homework/)
按章节组织，每个章节包含相应的编程练习：
- `chapter_11/` - 第十一章作业
- `chapter_12/` - 第十二章作业
- `chapter_13/` - 第十三章作业

### 实验 (experiments/)
按实验编号组织，每个实验包含多个程序：
- `experiment_1-basic_io/` - 实验1：基本输入输出
- `experiment_2-variable_operations/` - 实验2：变量操作
- `experiment_3-control_flow/` - 实验3：控制流程
- `experiment_4-functions/` - 实验4：函数
- `experiment_5-arrays/` - 实验5：数组
- `experiment_6-pointers/` - 实验6：指针
- `experiment_7-strings/` - 实验7：字符串
- `experiment_8-structures/` - 实验8：结构体
- `experiment_9-user_management/` - 实验9：用户管理系统

## 编译运行
所有文件均为标准C语言源文件，可使用gcc等编译器编译运行。

示例：
```bash
gcc experiments/experiment_1-basic_io/1-basic_io.c -o output
./output
```
