# 摩尔斯密码发报机软件

## 项目简介

本项目是一个电子工艺实习课程项目的软件部分，实现了一个基于嵌入式系统的摩尔斯密码发报机。软件运行在TI MSP微控制器上，通过OLED显示屏和按键进行交互，能够将用户输入的文本信息转换为摩尔斯码信号，并通过GPIO引脚控制外部设备（如LED或蜂鸣器）进行发送。

## 功能特性

- **摩尔斯码编码**：支持字母（a-z）和数字（0-9）的摩尔斯码转换
- **交互式配置**：通过OLED显示屏和按键设置发送内容、点划时长和发送间隔
- **信号控制**：提供标准的摩尔斯码开始信号（_._._）和结束信号（..._.）
- **循环发送**：可配置间隔时间，循环发送摩尔斯码信号

## 使用方法

### 1. 配置模式

系统启动后进入配置模式，用户可以通过按键设置以下参数：

1. **信息内容**（INFO）：选择要发送的字母或数字
2. **点划时长**（Duration）：设置点和划的基本时间单位（毫秒）
3. **发送间隔**（GAP）：设置两次发送之间的间隔时间（毫秒）

### 2. 按键功能

- **Prev键**：向前选择字符/数字
- **Enter键**：确认选择当前字符/数字
- **Next键**：向后选择字符/数字
- **Delete键**：删除已输入的字符

### 3. 发送模式

配置完成后，切换到发送模式，系统将按照设置的参数循环发送摩尔斯码信号。

## 摩尔斯码编码规则

- 点（dot）：GPIO高电平，持续 duration 毫秒
- 划（dash）：GPIO高电平，持续 duration × 3 毫秒
- 字符内间隔：GPIO低电平，持续 duration 毫秒
- 字符间间隔：GPIO低电平，持续 duration × 3 毫秒
- 单词间间隔：GPIO低电平，持续 duration × 7 毫秒

## 代码说明

### 主要函数

1. `char2index()`：将字符转换为摩尔斯码表索引
2. `dot()` / `dash()`：生成点和划信号
3. `sep()` / `char_sep()` / `word_sep()`：生成不同级别的间隔
4. `start_signal()` / `end_signal()`：生成开始和结束信号
5. `string2control()`：将字符串转换为摩尔斯码控制信号
6. `morse_cycle()`：完整的摩尔斯码发送循环
7. `main()`：主程序，处理用户交互和系统控制

### 数据结构

- `morse_table[]`：摩尔斯码查找表，使用'0'表示点，'1'表示划
- 配置参数存储在全局变量中，包括发送内容、点划时长和发送间隔

## 注意事项

- 确保硬件连接正确，特别是GPIO引脚配置
- 点划时长不宜设置过短，否则可能无法正确识别
- 发送间隔应考虑实际应用场景，避免过于频繁的发送
