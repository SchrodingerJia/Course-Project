# 数据结构课程实验项目

本项目为数据结构课程的实验代码集合，涵盖了链表、队列、栈、树、排序、动态规划和搜索等核心数据结构与算法主题。

## 项目结构

项目按实验主题组织，每个实验主题对应一个独立的目录，目录内包含该主题下的各个任务实现。

```
.
├── lab_1_linked_list_stack_queue/      # 实验一：链表、队列与栈
│   ├── 1_task_1_linked_list.c          # 任务1：链表操作
│   ├── 1_task_2_queue.c                # 任务2：队列操作
│   ├── 1_task_3_stack_queue.c          # 任务3：栈与队列综合应用
│   └── 1_task_4_reverse_string.c       # 任务4：字符串反转
├── lab_2_tree/                         # 实验二：树
│   ├── 2_task_1_binary_tree.c          # 任务1：二叉树构建与遍历
│   ├── 2_task_2_bst.c                  # 任务2：二叉搜索树操作
│   └── 2_task_3_tree_coloring.c        # 任务3：树着色问题
├── lab_3_sort_dp/                      # 实验三：排序与动态规划
│   ├── 3_task_1_knapsack.c             # 任务1：完全背包问题
│   ├── 3_task_2_number_mapping_sort.c  # 任务2：数字映射排序
│   ├── 3_task_3_string_frequency_sort.c # 任务3：字符串频率排序
│   └── 3_task_4_envelope_nesting.c     # 任务4：信封套娃问题
├── lab_4_search/                       # 实验四：搜索
│   ├── 4_task_3_maze.c                 # 任务3：迷宫问题
│   ├── metro_data.txt                  # 地铁网络数据
│   └── no2_metro_data.txt              # 地铁站点名称映射数据
└── README.md                           # 项目说明文档
```

## 功能概述

### 实验一：链表、队列与栈 (lab_1_linked_list_stack_queue)
- **任务1**：实现链表的基本操作（创建、插入、删除、遍历等）。
- **任务2**：实现队列的基本操作（入队、出队、判空等）。
- **任务3**：综合运用栈和队列解决实际问题。
- **任务4**：使用栈实现字符串反转功能。

### 实验二：树 (lab_2_tree)
- **任务1**：根据前序遍历和中序遍历序列重建二叉树，并进行层序遍历输出。
- **任务2**：实现二叉搜索树（BST）的插入、查找和删除操作。
- **任务3**：解决树着色问题，确保相邻节点颜色不同。

### 实验三：排序与动态规划 (lab_3_sort_dp)
- **任务1**：解决完全背包问题，计算在给定容量下的最大价值。
- **任务2**：根据数字映射规则对数组进行排序。
- **任务3**：根据字符串出现频率对字符串数组进行排序。
- **任务4**：解决信封套娃问题（俄罗斯套娃信封），计算最多能嵌套的信封数量。

### 实验四：搜索 (lab_4_search)
- **任务3**：解决迷宫问题，寻找从起点到终点的路径。
- **辅助数据**：包含地铁网络数据（`metro_data.txt`）和站点名称映射数据（`no2_metro_data.txt`），用于可能的图搜索问题。

## 使用方法

1. **环境要求**：
   - 本项目所有代码均使用C语言编写。
   - 需要支持C99标准的编译器（如gcc）。

2. **编译与运行**：
   - 进入对应实验目录，使用gcc编译C文件。
   - 示例：编译并运行实验二的任务1
     ```bash
     cd lab_2_tree
     gcc 2_task_1_binary_tree.c -o binary_tree
     ./binary_tree
     ```
   - 大多数程序从标准输入读取数据，请根据程序提示或代码逻辑输入相应格式的数据。

3. **输入输出说明**：
   - 每个C文件通常包含`main`函数，可以直接编译运行。
   - 输入格式在代码注释或运行时提示中说明，一般为先输入数组大小，再输入数组元素。
   - 输出格式为问题要求的计算结果，如排序后的数组、最大价值、路径等。
