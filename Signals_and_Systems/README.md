# 信号与系统实验项目

本项目包含一系列信号与系统课程的 MATLAB 实验，涵盖了信号分析、傅里叶变换、系统响应等核心内容。

## 项目结构

```
.
├── README.md               # 项目说明文档
├── lab_01/                 # 实验一：信号分析与傅里叶变换
│   ├── exp_1_double_sided_exponential.m          # 实验1：双边指数信号的时域和频域分析
│   ├── exp_2_triangular_pulse_spectrum.m         # 实验2：三角脉冲频谱的三种计算方法
│   └── exp_3_fourier_series_decomposition.m      # 实验3：周期矩形脉冲的傅里叶级数分解
└── lab_02/                 # 实验二：系统分析
    └── exp_12_system_response.m                  # 实验12：系统响应分析
```

## 实验内容简介

### 实验一 (lab_01)
1.  **exp_1_double_sided_exponential.m**: 绘制双边指数信号 `f(t) = e^{-|t|}` 的时域波形，并计算和绘制其傅里叶变换的幅度频谱。
2.  **exp_2_triangular_pulse_spectrum.m**: 使用三种方法（数值积分法、卷积定理法、理论计算法）计算三角脉冲信号的频谱，并进行对比展示。
3.  **exp_3_fourier_series_decomposition.m**: 对周期矩形脉冲信号进行傅里叶级数分解，计算其幅度谱和相位谱，并与原始信号对比。

### 实验二 (lab_02)
1.  **exp_12_system_response.m**: 给定系统传递函数和输入信号，计算并绘制系统的时域响应，同时输出特定时间点的响应值。

## 使用说明

1.  **环境要求**: 确保已安装 MATLAB 软件。
2.  **运行实验**: 进入对应的实验目录，直接在 MATLAB 中运行相应的 `.m` 脚本文件。部分实验（如实验3）运行时需要根据提示输入参数（如周期 `T` 和脉宽 `tao`）。
3.  **结果查看**: 每个脚本运行后会生成相应的图形窗口，展示信号波形、频谱或系统响应曲线。
